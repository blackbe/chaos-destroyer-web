# Book & Article Reading Tracker

Track books for reconnection research, confidence building, and personal growth.

## Usage

```bash
# Add a book to your reading list
node reading-tracker.js add "The Body Keeps the Score" "Bessel van der Kolk"

# Log a book as completed
node reading-tracker.js add "Hold Me Tight" "Sue Johnson"

# View your progress
node reading-tracker.js view
```

## Data Files

- `data/readings.json` - Books you've completed
- `data/books.json` - Books on your to-read list

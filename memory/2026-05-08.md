### System Architecture (Final)

**Build Pipeline:**
```
00:00 PDT (midnight) → Builder spawns
  ↓
Parse backlog, find first unblocked [ ] task
  ↓
Generate prompt for task (HugBack or CHAOS-DESTROYER)
  ↓
Call local Ollama llama3.2:3b (FREE, http://localhost:11434)
  ↓
Extract code files (auto or manual fallback)
  ↓
Git commit & push to repo
  ↓
Auto-deploy to Vercel (HugBack only)
  ↓
Mark [x] task complete in backlog
```
